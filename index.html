Crystals Collector!

Objective: Match your score with the target score.

There are four jewels. Each jewel contains a hidden value. Whenever a jewel is clicked, it will reveal the hidden value. The hidden points value represented by that jewel will then be added to your score.

If your score goes over the target score, you lose.

The value of each jewel will change when the game restarts.

Good luck!
